

## Fully functional linear regression model developed in Python 

### Project Overview 
* Implements linear regression from scratch in Python using NumPy
* Supports gradient descent, L2 Regularisation and feature normilisation
* Includes utilities for generating sample data for testing

### Features 
* Fit linear models using gradient descent
* Support for Learning rate, number of iterations and L2 regularisation
* Optional feature normalisation feature
* Stores loss history for analysis
* Predict new data once trained
